# Rashmi Shree Marketplace

This workspace now contains a starter blueprint for the Rashmi Shree fashion marketplace:

- Frontend: React + Vite + Tailwind + React Router + Zustand
- Backend: Cloudflare Workers + Hono
- Auth: Firebase Google / OTP hooks
- Data: PostgreSQL schema references and R2-ready service skeleton
- Docs: architecture and API blueprint

Use this as the foundation for the full 200+ file implementation.
